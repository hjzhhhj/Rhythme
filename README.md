# 🎵 Rhythme

> 악보 읽기가 어려운 피아노 초보자를 위한
> **귀여운 계이름 타임어택 웹 게임** 🎹✨

<br />

## 💡 Idea

오선지에 랜덤으로 나오는 30개의 음표를 키보드로 빠르게 입력하며
게임처럼 계이름을 익히는 **React 기반 음악 학습 서비스**입니다.

<br />

## 🎯 기획 배경

피아노를 배우면서 악보의 계이름을 읽는 속도가 느려
연습을 시작하기도 전에 지치고 흥미가 떨어지는 경우가 많았습니다.

그래서 악보 읽기를 어렵고 지루한 공부가 아니라,
**빠르게 보고, 누르고, 맞히는 게임 방식**으로 자연스럽게 연습할 수 있도록 기획했습니다.

<br />

## 🕹️ 서비스 소개

사용자는 오선지에 나타나는 랜덤 음표를 보고
키보드 `S ~ K` 키를 눌러 정답을 입력합니다.

정답을 맞히면 다음 음표가 등장하고,
오답을 입력하면 라이프가 감소합니다.

총 **30개의 음표를 얼마나 빠르게 맞히는지**를 기록하며
반복 플레이를 통해 자연스럽게 독보력을 키울 수 있습니다.

<br />

## ✨ 주요 기능

* 🎼 **랜덤 음표 생성**
  난이도에 따라 오선지 위에 랜덤 음표가 등장합니다.

* ⌨️ **키보드 입력 판정**
  사용자가 입력한 키가 정답인지 즉시 확인합니다.

* ❤️ **라이프 시스템**
  오답 입력 시 라이프가 감소하여 긴장감을 줍니다.

* ⏱️ **타이머 기능**
  30개의 음표를 모두 맞히는 데 걸린 시간을 측정합니다.

* 💾 **기록 저장**
  최고 기록과 최근 플레이 기록을 LocalStorage에 저장합니다.

<br />

## 🌟 기대 효과

* 악보의 계이름을 더 빠르게 읽을 수 있습니다.
* 반복 학습을 게임처럼 부담 없이 할 수 있습니다.
* 피아노 연습 전 가볍게 워밍업할 수 있습니다.
* 초보자의 악보 읽기 진입 장벽을 낮출 수 있습니다.

<br />
